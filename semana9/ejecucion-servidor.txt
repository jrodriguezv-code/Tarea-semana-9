Al ejecutar el comando 'node server.js', el entorno de ejecución de Node.js carga el framework Express y reserva el puerto TCP 3000 del sistema operativo. A partir de ese instante, el proceso se queda en estado de escucha activa en segundo plano, listo para interceptar cualquier solicitud HTTP entrante.

Cuando un cliente realiza una petición, el middleware de logs registra la marca de tiempo, el método (GET/POST) y la ruta solicitada directamente en la consola del servidor, procediendo a despachar el contenido programado para '/' o '/estado' con un código de estado HTTP 200 (Éxito).
